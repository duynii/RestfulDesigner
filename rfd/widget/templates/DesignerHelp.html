Drag n Drop

Drop resource into the designer here to create branch.
Double click a branch to delete it.
Right click on '/' button on URL element for context menu.

URI Element

Hover mouse over URI Element (colour-ised) to bring out configuration menu for element.
Hover mouse out of configuration menu to close it.

Click '/' button to create new sub-branch
Right click '/' button to bring context menu: 'Add new sub branch', 'Delete URI element'
Double URI element to delete it without context menu
