Drag n Drop Resources

Drop resource into the designer on the left to create a new URI branch.

URI Elements

Placeholder: A simple URI element that supports no method.
Static: A URI element to simply serve static URI data in the form of JSON e.g. contact_us page
Param: A custom resource, you can set as a placeholder URI element or a 'template' parameter input.
- e.g. some.com/{user_id}

Add any HTTP method and configure it.
Use to create URI Elements of archetypes Controller, Processing Function, Placeholder, and template parameter.
